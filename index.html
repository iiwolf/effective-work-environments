    Don’t see what you’re looking for?

  • These are just some examples that I find very useful, next time you’re “in the field”, think
    about what tools could make your life better:
    • “It would be cool if I could ______ “
    • “This is going to take forever, I wish I could just _______”
    • “Ugh, now I’m going to have to do that again, I wonder if there’s a way to _______"
  • Chances are, there’s a tool for that!
    • If not, there’s a chain of tools that you could use, turn into a function, and use to make your life easier.
    • If you don’t want to figure it out right now, start keeping a list of things you’d like to research for later

    Editor

    VSCode!

    Microsoft does something very right

    VSCode Setup

  • Workspaces - contains project specific settings
    • Applicable folders - Defines search space (CTRL+X+F)
    • Language
    • Debugger
    • Default toolchain for any other operations
  • Keybindings (keyboard shorcuts)
    • Navigating Workspace: splitting windows, switching windows, jumping, opening sidebar
    • Navigating File: searching, find-and-replace, jumping blocks, goto line
  • Extensions
    • Everything else!

    VSCode extension recommendations

    • Workflow Support
      • Your favorite keybinding package (e.g. Emacs or Vim)
      • Remote - ssh, WSL, and even containers
    • Tool Support
      • GitLens - Git supercharged
      • Excel Viewer - view csv's
      • CMake - compile, intellisense, etc.
      • Markdown All in One - intellisense, preview, etc.
      • LaTeX Workshop - compile, intellisense, preview, etc.
      • Docker - Create, manager, and debug containers
    • Language Support
      • Python - Pylance, Python Docstring Generator
      • C/C++ - C/C++ Intellisense
      • Fortran - Modern Fortran
      • ...
